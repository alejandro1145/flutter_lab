¿Qué es Flutter?

Flutter es un SDK creado por Google para desarrollar aplicaciones móviles, web y escritorio usando un solo código.

SDK

Flutter incluye herramientas para crear aplicaciones modernas.

Comandos eficientes

Permite desarrollar aplicaciones de manera rápida y sencilla.

Desarrollo móvil

Compatible con Android, iOS y Web desde un solo proyecto.

Dart

Lenguaje utilizado por Flutter para aplicaciones rápidas.

Widgets

Todo en Flutter se construye mediante widgets.

StatefulWidget

Widgets dinámicos que cambian durante la ejecución.

StatelessWidget

Widgets estáticos que no cambian su información.

¿Por qué usar Flutter?

Flutter permite crear aplicaciones modernas, rápidas y visualmente atractivas usando un solo código fuente.
